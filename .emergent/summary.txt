<analysis>**original_problem_statement:**
The user wants to build a comprehensive CRM and sales funnel automation platform for a brand called Fakulti.

PRODUCT REQUIREMENTS:
- **AI Agent Identity:** An AI virtual assistant for Fakulti Labs to handle incoming leads, qualify them, generate quotes and PDF orders, close sales, and manage customer loyalty. The tone should be human-like, and it must avoid making medical claims.
- **Intelligent Funnel:** An automated system that categorizes leads into different stages (e.g., New, Interested, Hot, Client, Cold) based on their behavior (keywords, response time, game interaction, etc.).
- **CRM Panel:** A full-featured admin dashboard to manage the system.
    - **Dashboard:** Displaying total leads, leads per stage, sales, and conversion rates.
    - **User Panel:** A detailed view of each user with fields like name, WhatsApp, contact info, product interest, lead source, game history, purchase history, etc.
    - **Bulk Operations:** Ability to upload leads from an Excel file (.xlsx) and download filtered lead lists.
- **Gamification:** Integration of games (Roulette, Mystery Box, Lucky Button) into the CRM. Game interactions should automatically update lead data, stage, and prize information. The user later requested to replace the Mystery Box and Lucky Button with a Slot Machine and a Scratch Card game.
- **Loyalty System:** A configurable post-sale messaging sequence (up to 24 messages) to engage customers, provide educational content, and encourage repurchases.
- **Automation Panel:** A new panel to manage all the rules the bot must follow when interacting with a customer.
- **Real WhatsApp Connection:** Integration with the official WhatsApp Cloud API via Meta for Developers to allow the bot to send and receive real messages.
- **Human Agent Handover:** A mechanism to pause automation and notify a human agent when requested by the user or for special cases.
- **Scalability:** The system must be designed to easily add new products, games, and rules without major structural changes.
